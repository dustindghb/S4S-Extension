üìã Setup Instructions

1. Install Ollama (Optional)

For local AI analysis, install Ollama and the Llama 3.2 Vision model:

curl -fsSL https://ollama.ai/install.sh | sh
ollama pull llama3.2-vision
ollama serve

2. Configure Extension

Open the extension options and configure:

  • OpenAI API key (required)
  • Ollama server URL (default: http://localhost:11434)
  • Ollama model name (default: llama3.2-vision)
  • Enable grayscale processing (recommended)
  • Custom prompts for each model

3. Test Connections

Use the test buttons in options to verify:

  • OpenAI API connection and vision model access
  • Ollama server connection and model availability
